## Installation

### Dependency

#### Operating System

Windows is not supported currently.

#### Software modules

pytorch >= 1.7
gcc >= 10.2

#### Simple install 

```
git clone --recursive git@github.com:cyyever/cbuild.git
cd cbuild
bash setup.sh
./cbuild cyy_naive_pytorch_lib:master    
```
